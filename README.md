# vimfiles

Portable vim configuration from Manuel Santos

## Install

For windows change `.vim` form `vimfiles` accordingly.
```
git clone git://github.com/undo76/vimfiles ~/.vim
cd ~/vimfiles
./install.sh
```

## Uninstall
```
cd ~/.vim
./uninstall.sh
```
